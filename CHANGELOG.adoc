== [0.1.0] - 2026-03-21

=== Phase 1 Complete

- Distribution parser (Beta, Dirichlet, Categorical distributions)
- Ternary codegen (true/false/unknown probabilistic states)
- Probabilistic model generation from manifest specs
- Risk model example with full pipeline
- Bespoke manifest parsing for probabilistic modelling
- Integration tests for distribution parsing and codegen

=== Added (scaffold — 2026-03-20)

- Initial project scaffold from rsr-template-repo
- CLI with subcommands (init, validate, generate, build, run, info)
- Manifest parser (`betlangiser.toml`)
- Codegen engine stubs
- ABI module (Idris2 proof type definitions)
- Library API for programmatic use
- Full RSR template (17 CI workflows, governance docs, bot directives)
- README.adoc with architecture overview
